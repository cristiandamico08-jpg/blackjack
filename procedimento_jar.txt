creare nella cartella bin un file chiamato MANIFEST.MF ed all'interno inserire questo codice (inserire una riga vuota alla fine del file):
Manifest-Version: 1.0
Main-Class: Main
Class-Path: lib/javafx.base.jar lib/javafx.controls.jar lib/javafx.fxml.jar lib/javafx.graphics.jar ...


----------------------------------------------------------------------------------------------------

dopo aver compilato i file, eseguire in un terminale bash il seguente comando:
jar cfm BlackJack.jar bin/MANIFEST.MF -C bin .

----------------------------------------------------------------------------------------------------

dopo aver creato il file avviare il file avvia.vbs
